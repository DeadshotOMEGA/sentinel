# Kiosk Check-In

> Audience: Duty Day Staff (DDS), kiosk operators

## Purpose

Use kiosk mode for efficient, supervised member check-in flow.

## Procedure

1. Confirm kiosk station readiness.
2. Prompt member to scan badge.
3. Validate success state in kiosk response.
4. Confirm presence update in dashboard/presence grid.
5. Escalate scanner failures to badge administration workflow.
