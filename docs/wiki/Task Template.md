# Check In a Member

## What this does
Registers a member as present for the day.

## When to use it
Use this when a member arrives at the unit and needs to be marked present.

## Steps

1. Open the Sentinel Dashboard
2. Click **Members**
3. Click **Check In**
4. Scan the member's RFID card

## Result
The member will appear in the **Active Members** list.
